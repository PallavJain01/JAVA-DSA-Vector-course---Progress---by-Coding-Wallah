### Type Casting
- Automatic Type-Casting (Wide)
```java
newType newVar = oldVar;

double db = 10.45F;
```

- Manual Type-Casting (Narrow)
```java
newType newVar = (newType) oldVar;

int num = (int) 123456789010L;
```
